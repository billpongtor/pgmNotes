simplified use of init, add, and commit


--------------------------------------------------------
'git' basically has 3 steps which are used to begin controlling file versioning.

step 1:
git init is used to create the file repository. by default, it creates a folder named .git in the current working directory
adding files to a repository is a 2 step process:

step 2a:
use git add command to add files to a staging area
step 2b:
use git commit command to move the files from the staging area to the repository

the above steps are the very basic actions involved to create and add files to a repository
