git 'add' command used to stage files to be commited to a repository


The git command 'git add' is the prelimnary step used to stage files that will be eventually
commited to a repository (usually a local repository folder named .git)
Using 'git add .' will stage ALL files in the project folder
Using 'git add etc,... explicitly designates specific files.
