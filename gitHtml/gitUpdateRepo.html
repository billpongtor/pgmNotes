updating the repository with modified and changed files


The git command 'git status' will indicate any tracked and untracked files.
New files will be designated as untracked, files with changes will be marked as 'modified'.
Using 'git add filename filename etc,...' will pre-stage these files to be commited to the repo.
Then the 'git commit -m ""' command is issued to place these files into the repo.
