vi setup notes

Usually, the users home directory contains a file called .vimrc.
.vimrc is used to add user favorite vi customizations such as changing the tab spacing etc,...

Common .vimrc entries and their effect on vi:

adding comments to .vimrc file is as simple as preceding the comment line with a single double quote:
example: " this is the comment format using a single double quote preceding the comment string

changing the tab spacing to a user preferred value:
set tabstop=number : tabstop=number changes the number of spaces a tab uses
example: set tabstop=4 changes the tabspacing to use 4 spaces
